# Shekhar Singh - Portfolio Website

A modern, responsive portfolio website built with pure HTML, CSS, and JavaScript (Tailwind CSS via CDN).

## Features

- ✅ **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- ✅ **Dark Theme** - Modern dark color scheme with emerald accents
- ✅ **Smooth Navigation** - Smooth scrolling between sections
- ✅ **Interactive Contact Form** - Form validation with user feedback
- ✅ **Hover Effects** - Beautiful hover animations on cards and buttons
- ✅ **Sticky Header** - Header stays visible while scrolling
- ✅ **Accessible** - Semantic HTML elements and proper ARIA labels

## Sections

1. **Hero** - Introduction with availability status and stack overview
2. **About** - Personal information and key focus areas
3. **Projects** - Showcase of work with tech stack tags
4. **Skills** - Technical skills organized by category
5. **Contact** - Contact form and social links
6. **Footer** - Copyright and credits

## How to Use

Simply open `index.html` in a web browser. No build process or installation required.

### For Deployment

You can deploy this single HTML file to any static hosting service:
- Netlify
- Vercel
- GitHub Pages
- Any web server

## Technologies

- **HTML5** - Semantic markup
- **CSS3** - Styling with Tailwind CSS (via CDN)
- **JavaScript (ES6+)** - Interactive functionality
- **Tailwind CSS** - Utility-first CSS framework

## Customization

To customize the portfolio:

1. **Personal Info** - Update name, email, and social links in the HTML
2. **Projects** - Modify the projects section with your own work
3. **Skills** - Update the skills list to match your expertise
4. **Colors** - Modify Tailwind classes to change the color scheme
5. **Content** - Replace all text content with your own
